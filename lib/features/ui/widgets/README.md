# UI Widgets

Reusable UI components for the game.

## Purpose
- Build modular, reusable widgets
- Maintain consistent UI across screens

## Files
- `dice_card.dart` - Card-style die display widget
- `score_sheet.dart` - Score sheet categories widget
- `game_controls.dart` - Roll and play button controls
- `game_info_header.dart` - Header with score and game info
